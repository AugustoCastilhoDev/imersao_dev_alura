Projeto Coração Vascaíno

Sobre o Projeto

"Coração Vascaíno" é uma página web interativa e informativa dedicada a todos os torcedores e admiradores do Club de Regatas Vasco da Gama. 
O projeto foi desenvolvido como uma central de informações, reunindo dados históricos, informações sobre o elenco atual, produtos e muito mais em uma interface amigável e de fácil navegação.

A aplicação é construída com tecnologias web fundamentais e demonstra a manipulação dinâmica do DOM para criar uma experiência de usuário rica e responsiva.

Tecnologias Utilizadas
Este projeto foi construído utilizando as seguintes tecnologias:

HTML5: Para a estrutura semântica da página.
CSS3: Para estilização, layout e design responsivo, incluindo o uso de Google Fonts para uma tipografia personalizada.
JavaScript (Vanilla): Para toda a lógica da aplicação, incluindo a busca de dados, renderização dinâmica de conteúdo, interatividade e manipulação de eventos.
JSON: Utilizado como um banco de dados local para armazenar e fornecer todas as informações exibidas na página, como detalhes de jogadores, fatos históricos e produtos.

Funcionalidades Principais

Busca Dinâmica: O usuário pode pesquisar por termos específicos na barra de pesquisa para filtrar e encontrar informações rapidamente.

Navegação por Filtros: O menu de navegação permite filtrar o conteúdo por categorias, como "História", "Títulos", "Ídolos", "Elenco", "Notícias" e "Loja".

Renderização de Cards: As informações são apresentadas em cards gerados dinamicamente a partir do arquivo data.json, organizando o conteúdo de forma visualmente agradável.

Carrossel de Jogadores: O elenco é exibido em um carrossel interativo, permitindo que o usuário navegue horizontalmente para ver os jogadores.

Links Externos: Cada card contém um link que direciona o usuário para a fonte original da informação, seja o site oficial do Vasco, páginas de notícias ou lojas.
